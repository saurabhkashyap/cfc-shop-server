
## Installation

```bash
$ npm install
```

## Module Descriptions
 - coupon: Discount group with basic outfit information to group discount/offer codes.
 - code: Customer/public facing code to redeem.
 - product: List products, pricing and capacity.
 - invoice: Manage invoice state


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
